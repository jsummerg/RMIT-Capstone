## Installation
[(Back to top)](##table-of-contents)
Step 1: run the following in the console
```
npm install --legacy-peer-deps
```
Step 2: Pick server
- For production server
```
npm run build-prod
npm run start
```
 - For dev server:
```
npm run build-dev
npm run start
```
 - For live dev server:
```
npm run build-dev-live
```
Step 3: To perform tests with jest
```
npm run test
```
